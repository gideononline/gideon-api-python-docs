GIDEON ID Codes¶

gideon_api.lookup_item(category, item)¶

Looks up the GIDEON ID for a particular item.

Parameters

  • category (str) – The GIDEON API to search from such as diseases, vaccines, countries, etc. Refer to the gideon_api.get_endpoint() function documentation for a complete list.

  • item (str) – The name of the item, such as a particular disease or bacteria.

Return type

Union[int, str, None]

Returns

If the item is found, the GIDEON API code

gideon_api.get_endpoint(category)¶

Checks the category refering to a GIDEON API piece with a unique identifier.

Parameters

category (str) – One of the following: ‘diseases’, ‘drugs’, ‘vaccines’, ‘agents’, ‘vectors’, ‘vehicles’, ‘reservoirs’, ‘bacteria’, ‘mycobacteria’, ‘yeasts’, ‘countries’, ‘regions’

Return type

str

Returns

A string representing the API endpoint for the particular category of items in the GIDEON API
